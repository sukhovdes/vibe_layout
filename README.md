# Ozon Банк · B2B Лейаут для дизайнеров

Живой шаблон страницы, собранный 1:1 из Figma (файл «Прототип», node `82167-3114`).
Открывается в браузере, настраивается тумблерами — чтобы все понимали, как
организовать макет, без путаницы в Figma.

## Как запустить
```bash
python3 -m http.server 4188
```
Открыть http://localhost:4188 (или просто открыть `index.html` в браузере).

## Панель «⚙️ Настройки лейаута» (справа сверху)
- **Состояние контейнера** — дропдаун: `Заполнено / Пустое / Загрузка / Ошибка`.
- **Блоки страницы** — тумблеры: левое меню, степпер, кнопка Action, кнопка «Назад»,
  иконка-инфо, уведомления, плашка пользователя.
- **Настройки страницы** — текст заголовка и ширина (`Desktop XXL` / резиновая).

## Структура файлов
```
index.html        каркас лейаута + панель управления
styles.css        дизайн-токены из Figma + все стили
layout.js         логика панели (+ хук Layout.onReady для анимаций)
assets/icons/     иконки, выгруженные из Figma (SVG, 24px фрейм)
```

## Иконки
Все иконки взяты **только из Figma** и лежат в `assets/icons/` как SVG в 24px-фрейме
(`ic_s_*` — 16px). Аватар — фото-PNG.
Если ассета в Figma нет (например, иллюстрации для пустого экрана/ошибки) —
на его месте стоит **серый круг-плейсхолдер** (`.placeholder-circle`).
Экспортируй нужную картинку из Figma и замени круг.

## Подключение библиотек (анимации, JS)
1. В самом низу `index.html` раскомментируй нужную строку — GSAP / anime.js / AOS.
2. Код пиши в `layout.js` внутри хука:
```js
Layout.onReady(function () {
  gsap.from('.content-container', { opacity: 0, y: 20, duration: 0.4 });
});
```
`onReady` срабатывает после сборки страницы — библиотека уже доступна.

## Дизайн-токены
Все цвета/радиусы/типографика — CSS-переменные в `:root` (`styles.css`),
названия совпадают с Figma Variables (`--bgPrimary`, `--textAction`, `--radius-600` и т.д.).
Шрифт — **Onest** (подключён с Google Fonts).
